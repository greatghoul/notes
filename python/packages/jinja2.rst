控制语句不产生空行
~~~~~~~~~~~~~~~~~~
通常，使用 {% ... %} 这样的控制语句会产生一些空行，虽然以于用户来说，这些空行对网页的功能并没有什么影响，但一些有洁癖的人来说，是不能容忍的。 Jiaja2 提供了忽略空行的方法

.. code-block:: jinja

    {% for i in some_array -%}
        do something with "{{ i }}"
    {% end for -%}
